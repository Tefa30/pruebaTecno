¿QUE ES TECNOGUIAS?

Tecnoguias es una plataforma educativa enfocada en la tecnología. La página presenta actividades relacionadas con la instalación y configuración de distribuciones Linux, scripts en Bash, uso de Docker y Kubernetes, y el desarrollo de sitios web para presentar entregables. Incluye instrucciones, recursos y criterios de evaluación, con énfasis en fortalecer habilidades en entornos Linux, contenedores y orquestación para aplicaciones reale

Objetivo General:

Facilitar el aprendizaje práctico y autodidacta en Linux, automatización con Bash, y el manejo de Docker y Kubernetes mediante recursos visuales y explicativos claros.

Objetivos Específicos:

  • Guiar a los usuarios en la instalación y configuración de diferentes distribuciones de Linux con instrucciones detalladas.
  • Enseñar técnicas de automatización y scripting con Bash a través de tutoriales paso a paso
  • Mostrar cómo utilizar Docker para crear, gestionar y desplegar contenedores de forma sencilla.
  • Explicar la gestión y orquestación de aplicaciones con Kubernetes de forma comprensible y práctica.
  • Complementar todo el contenido con videos, capturas y explicaciones claras que faciliten el entendimiento y la práctica

    INSTALAR CENT0X

    Se explica cómo instalar CentOx, configurando la máquina virtual, asignando memoria y disco, y conectando a internet. Se crean usuarios y se instalan programas esenciales como sudo, make, rsync y git. También se mencionan editores de código como nano, finalizando con la invitación a seguir aprendiendo sobre scripts.
